## Development

Getting start is easy. Just clone the repository and run `make run` (or `.\make.ps1 run` in Windows PowerShell).

```bash
$ git clone https://github.com/tareq-khalil/HAWKTHORNE-CENTER.git
$ cd HAWKTHORNE-CENTER
$ make run
```
